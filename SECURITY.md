# Security Policy

## Supported Versions
We support the latest release and the `main` branch.

## Reporting a Vulnerability
We use GitHub Private Vulnerability Reporting (PVR) as the preferred channel.

- **Preferred:** https://github.com/devhms/nightshade/security/advisories/new
- **Fallback email:** security@example.com (replace this address before release)

Please do not open public issues for security vulnerabilities.

## What to Include
- Clear description of the issue
- Steps to reproduce
- Potential impact
- Any proof-of-concept or patches

## Response Targets
- Initial acknowledgement within 72 hours
- Fix or mitigation plan within 14 days when possible

## Disclosure
We coordinate fixes and may credit reporters unless you request anonymity.
